# Super Fitness App

A proprietary Flutter-based fitness application designed to help users track their workouts, manage their fitness goals, and maintain a healthy lifestyle. This application is for internal use only and is not available for public distribution.


## 📱 Features

- Workout tracking and management
- Goal setting and progress monitoring
- Localization support (multiple languages)
- Offline data storage using Hive and ObjectBox
- Secure data storage
- Image compression and management
- Speech-to-text functionality
- Network connectivity checking
- Beautiful UI with animations and loading states
- Cross-platform support (Android, iOS, Windows)

## 🛠️ Tech Stack

- **Framework**: Flutter
- **State Management**: Flutter Bloc
- **Local Storage**: Hive, ObjectBox, SharedPreferences
- **Networking**: Dio, Retrofit
- **UI Components**: 
  - Lottie for animations
  - Shimmer for loading states
  - Google Fonts for typography
  - Flutter SVG for vector graphics
- **Localization**: Easy Localization, Flutter Localization
- **Security**: Flutter Secure Storage
- **Image Handling**: Cached Network Image, Image Picker, Flutter Image Compress
- **Utilities**: 
  - Permission Handler
  - URL Launcher
  - Share Plus
  - Internet Connection Checker
