# 💪 Super Fitness App

A proprietary **Flutter-based fitness application** designed to help users track workouts, manage
goals, and maintain a healthy lifestyle.
> ⚠️ **Internal Use Only** – This app is not available for public distribution.

---

## 📱 Features

- 🏋️ Workout tracking and management
- 🎯 Goal setting and progress monitoring
- 🌐 Localization support (multi-language)
- 📦 Offline data storage: Hive & ObjectBox
- 🔐 Secure data handling
- 🖼️ Image compression & management
- 🗣️ Speech-to-text functionality
- 🌐 Network connectivity checking
- ✨ Beautiful UI with animations & loading states
- 💻 Cross-platform: Android, iOS, Windows

---

## 🛠️ Tech Stack

**Framework:** Flutter  
**State Management:** Flutter Bloc

### 📦 Local Storage

- Hive
- ObjectBox
- SharedPreferences

### 🌐 Networking

- Dio
- Retrofit

### 🎨 UI Components

- Lottie (animations)
- Shimmer (loading effects)
- Google Fonts
- Flutter SVG

### 🌍 Localization

- Easy Localization
- Flutter Localization

### 🔐 Security

- Flutter Secure Storage

### 🖼️ Image Handling

- Cached Network Image
- Image Picker
- Flutter Image Compress

### ⚙️ Utilities

- Permission Handler
- URL Launcher
- Share Plus
- Internet Connection Checker
